# Flowdash Web (MVC)

Flowdash is a Project Management System (PMS) designed to help teams organize projects, manage tasks, and collaborate efficiently.

## 🚀 Features
- User authentication (Login / Register)
- Project creation & joining
- Role-based access control (Owner, Admin, Member)
- Task boards (Kanban-style)
- Task management (create, update, reorder)
- Announcements & activity logs

## 🛠️ Tech Stack
- ASP.NET Core MVC (.NET 7)
- Entity Framework Core
- SQL Server
- HTML, CSS, JavaScript (jQuery, Bootstrap)
